## Simple React BMI Calculator

This project is as an exercise for learning useState. A simple form that takes in the values of weight and height with 2 buttons; one to calculate BMI and the second button to reset the values
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
